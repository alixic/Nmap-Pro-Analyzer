


# Nmap Analyzer Pro

<br>Denumire proiect in cadrul materiei: g2p17_RezultateNmap <br/>
Aplicație web pentru vizualizarea rezultatelor scanărilor Nmap în format XML.

## Funcționalități
- Upload fișier XML generat de Nmap
- Vizualizare hosturi, porturi, OS, MAC address, vendor
- Filtrare și căutare
- Paginare

## Utilizare
Deschide `index.html` cu orice aplicatie client de tipul browser.
